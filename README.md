# Activepieces Workflow Collection

This repository contains a collection of JSON workflow files for Activepieces, a workflow automation tool. These workflows are designed to automate various tasks and processes.

## Categories

### AI Agents & Assistants
*   **AI Newsletter**: Automated newsletter generation using AI.
*   **Automation Documentation Generator**: Generates documentation for workflows.
*   **General Applicant Database - OpenRouter**: Processes applicant data using OpenRouter AI.
*   **Generate System Prompt**: Helper workflow to create system prompts.
*   **Literature Review Writing Agent**: Agent for writing literature reviews.
*   **PostgreSQL Assistant**: AI assistant for database queries and management.
*   **Retrieve and Summarize Article Content**: Summarizes web articles.
*   **Smart Narrative Assistant**: Assists in creating narratives from data.
*   **Social Media Post Generation**: Generates content for social media.

### Data Reporting & Analytics
*   **Daily KPI Trend Chart → Slack**: Sends KPI trends to Slack daily.
*   **Data Reporting Newsletter**: Generates newsletters based on data reports.
*   **Google Analytics Report**: Automates Google Analytics reporting.
*   **Health System Daily Update**: Daily updates on health system metrics.
*   **Health System Monthly Bulletin**: Monthly bulletins for health system data.
*   **Indicator Loading Report**: Reports on data indicator loading status.
*   **Monthly Link Validation**: Validates links in reports monthly.
*   **Situation Report Retrieval**: Retrieves situation reports.
*   **Weekly Uptime Report**: Reports on system uptime weekly.

### Data Processing & ETL
*   **Consultancy Database**: Manages consultancy data.
*   **Data Entry Metadata Extraction**: Extracts metadata from data entries.
*   **PDF Datasource Table Extractor**: Extracts tables from PDF datasources.
*   **Stripe Charge → QuickBooks**: Syncs Stripe charges to QuickBooks.
*   **Webscraper**: General web scraping workflow.

### Document & File Management
*   **Convert html to pdf and save to drive**: Converts HTML content to PDF and saves it.
*   **Download email pdf attachments then convert to Images**: Processes email attachments.

### Marketing & Communications
*   **External Newsletter**: Manages external newsletters.
*   **Internal Newsletter**: Manages internal newsletters.
*   **Retrieve Mailchimp Subscribers**: Syncs Mailchimp subscribers.
*   **Send External Newsletter**: Automates sending external newsletters.
*   **Send Personal Morning Briefing**: Sends a personalized daily briefing.
*   **Team Blogposts**: Automates team blog post management.

## Usage
To use these workflows, import the JSON files into your Activepieces instance.
